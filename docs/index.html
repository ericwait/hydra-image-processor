🔨 Building the Documentation

Prerequisites:
You need to have Doxygen installed on your system.

Installation

Ubuntu/Debian:

sudo apt-get install doxygen graphviz

macOS:

brew install doxygen graphviz

Windows:

Download and install from doxygen.nl/download.html

Generate Documentation

From the root directory of the project, run one of these commands:

Using the build script (Linux/macOS):

./build_docs.sh

Using Doxygen directly:

doxygen Doxyfile

After building, the documentation will be available at:

docs/html/index.html
📖 View Documentation (after building)

📑 What's Documented

  • CUDA C++ API - Core image processing functions and classes
  • Image Containers - ImageView and ImageOwner classes
  • Vector Math - 3D vector operations and utilities
  • Device Management - CUDA device queries and memory management
  • Interface Layers - MATLAB MEX and Python bindings
